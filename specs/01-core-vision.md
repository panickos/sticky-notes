# Spec 01: Core Vision

**Status:** ✅ Approved  
**Depends on:** —

## Problem

Need quick, persistent reminders and scratch notes visible while working in other applications — without switching context or opening a separate notes app.

## Primary user

- **Solo** — personal productivity only (not team/shared)

## Solution summary

A macOS menu-bar utility that displays literal sticky notes floating above all other windows. Notes can be positioned anywhere on screen, colored, and edited with markdown. The entire note set can be hidden/shown via global hotkey.

## Success criteria (v1)

- **Polished UX** — feels like a real Mac app, not a rough prototype
- Suitable for daily personal use

## Explicitly not the goal (v1)

See [07-v1-boundaries](./07-v1-boundaries.md).

## Verification

- [x] Solo personal tool
- [x] Floating overlay notes over all apps
- [x] Show/hide entire note set
- [x] Local-only (no cloud/account in v1)
